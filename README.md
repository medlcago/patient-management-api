# Patient Management API

This is a RESTful API built with FastAPI for patient management.

### Installation and Deployment

1. Navigate to the `/backend` directory: `cd backend`
2. Create an `.env` file based on the provided `.env.example` file and populate it with the necessary environment variables.
3. Start the application using Docker Compose: `docker-compose up -d`
4. The API will be available at `http://localhost:8080`

### API Documentation

The API documentation is automatically generated by Swagger UI. You can access it at `http://localhost:8080/docs`
